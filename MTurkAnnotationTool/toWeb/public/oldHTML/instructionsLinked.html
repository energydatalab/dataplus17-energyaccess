General Instructions

Welcome! In this HIT, we will ask you to mark several objects in an image. We will be asking you to annotate objects that would indicate whether or not a community has access to electricity. Please follow the instructions below:

  1. Read these instructions carefully. When you have completed them, click the "Annotate" button at the bottom of the page. This will take you to a page where you can annotate an image. These instructions are the same every time you complete this task, however, so if you have done this task multiple times, feel free to skip.
  2. You will see a table at the top of the screen which will tell you what and how to annotate. The rows of the table will be:
    1. Sample image of what the object looks like
    2. Name of Object
    3. Tool to Annotate that object

    See the Example Below:



  3. Select the polygon tool and draw polygons around any objects you see that match the first sample image.
  4. Select the line tool and draw a line where youse see anything which matches second sample image.
  5. Select the label tool and label everything that resembles the labeled object in the third sample image.
  6. Hit submit once you have finished your annotations.

Using the Tools

Polygon

  1. Click on the edge of the object you would like to outline.
  2. Click around the perimter of the object you wish to outline
  3. Click on the first vertex to close the polygon
  4. Your polygon will turn from cyan and red to purple and blue because it is complete
  5. You will be allowed to switch tools
  6. Delete the last polygon you created with "Q"

Line

  1. Click at the start of the line you would like to draw
  2. Keep clicking along the path you wish to draw
  3. Hit "L" to complete the line once you have finished outlining it.
  4. Your line will turn from cyan and red to orange and green because it is complete.
  5. You will be allowed to switch tools
  6. Delete the last line you created with "W"

  3. Hit "L" to complete the line once you have finished outlining it.
  4. Your polygon will turn from cyan and red to purple and blue because it is complete.
  5. You will be allowed to switch tools
  6. Delete the last point you labelled with "E"

Other Notes

  • If you mess up the annotation you are currently working on, hit "R" to reset
  • You will be unable to change your annotation type while in the middle of making a line or polygon
  • Completed polygons will show up purple and blue. Completed lines will show up orange and green. Completed points will be hot pink
